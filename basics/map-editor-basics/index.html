Map editor basics

After opening a project, you should have this view:

Map selector

Take a look on the map selector (1). You can add, edit, remove, copy, and paste folders or maps here. We recommand you to use folders to sort your maps for each zones of your game. By default, there's also a Battle maps folder that you can use to put all the maps that you will use for battle backgrounds. It is not required to put these maps into this specific folder. Clicking on a map will load this one on the map editor (5). Right clicking will open this context menu:
